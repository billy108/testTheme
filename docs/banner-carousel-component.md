# Banner Carousel 组件使用文档

## 1. 组件概述

Banner Carousel 是一个功能强大的响应式轮播组件，专为 Shopify 主题设计，支持自动轮播、动态指示器和点击跳转等功能。

## 2. 功能特性

### 2.1 核心功能
- ✅ **自动轮播**：支持配置自动切换速度
- ✅ **动态指示器**：橙色填充进度条，直观显示当前幻灯片状态
- ✅ **点击跳转**：每张幻灯片可配置独立的跳转链接
- ✅ **响应式设计**：适配桌面端和移动端
- ✅ **自适应高度**：根据图片尺寸自动调整高度
- ✅ **完整图片显示**：移动端确保图片完整显示，不裁剪

### 2.2 视觉特性
- **橙色动态填充指示器**：平滑的进度条动画
- **响应式布局**：适配各种屏幕尺寸
- **平滑过渡效果**：幻灯片切换时的淡入淡出动画
- **适配不同图片尺寸**：支持多种宽高比图片

## 3. 安装方法

### 3.1 主题集成
1. 将 `banner-carousel.liquid` 文件上传到 Shopify 主题的 `sections` 目录
2. 在主题编辑器中添加 "Banner Carousel" 部分
3. 配置幻灯片内容

### 3.2 手动添加到模板
```liquid
{% section 'banner-carousel' %}
```

## 4. 配置选项

### 4.1 全局设置

| 设置项 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| Auto rotate slides | 布尔值 | true | 是否自动轮播幻灯片 |
| Change slides every | 数值 | 5 | 自动轮播间隔（秒），范围：3-90秒 |
| Image behavior | 选择 | none | 图片动画效果（none/ambient） |
| Accessibility label | 文本 | Banner carousel | 屏幕阅读器描述文本 |

### 4.2 幻灯片设置

| 设置项 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| Slide name | 文本 | Slide | 幻灯片描述名称 |
| Slide link | URL | - | 点击幻灯片跳转的链接 |
| Desktop Image | 图片 | - | 桌面端显示的图片 |
| Mobile Image | 图片 | - | 移动端显示的图片 |
| Image overlay opacity | 数值 | 0 | 图片覆盖层透明度，范围：0-100% |

## 5. 图片尺寸建议

### 5.1 桌面端图片
- **推荐尺寸**：1920 x 1080px (16:9)
- **最小尺寸**：1600 x 900px
- **最大尺寸**：3840 x 2160px
- **格式**：JPG, PNG, WEBP
- **最大文件大小**：2MB

### 5.2 移动端图片
- **推荐尺寸**：750 x 1000px (3:4)
- **最小尺寸**：600 x 800px
- **最大尺寸**：1920 x 2560px
- **格式**：JPG, PNG, WEBP
- **最大文件大小**：2MB

## 6. 使用示例

### 6.1 基本配置
1. 添加 Banner Carousel 部分到主题
2. 启用自动轮播
3. 设置轮播间隔为5秒
4. 添加2-5张幻灯片
5. 为每张幻灯片配置图片和链接

### 6.2 最佳实践

#### 6.2.1 图片优化
- 使用WebP格式以减小文件大小
- 确保图片分辨率足够高，避免模糊
- 保持图片主题一致，提升品牌识别度
- 为移动端和桌面端分别准备合适尺寸的图片

#### 6.2.2 内容策略
- 保持幻灯片数量在2-5张之间，避免过多影响加载速度
- 每张幻灯片突出一个主题或促销信息
- 使用清晰的CTA（Call to Action）文本
- 确保链接与幻灯片内容相关

#### 6.2.3 性能优化
- 控制图片文件大小，每张不超过2MB
- 合理设置轮播间隔，建议3-7秒
- 考虑在移动设备上禁用自动轮播，提升用户体验

## 9. 常见问题

### 9.1 幻灯片不显示
- 检查是否添加了图片
- 检查图片尺寸是否符合要求
- 查看浏览器控制台是否有错误

### 9.2 移动端显示异常
- 确保上传了合适尺寸的移动端图片
- 检查图片宽高比是否合理
- 清除浏览器缓存后重试

### 9.3 自动轮播不工作
- 检查是否启用了自动轮播设置
- 检查轮播间隔设置是否合理
- 查看JavaScript控制台是否有错误

### 9.4 指示器不显示
- 确保添加了至少2张幻灯片
- 检查CSS样式是否被覆盖
- 查看浏览器控制台是否有错误
